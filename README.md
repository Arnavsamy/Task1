Investor Scout Pro
Investor Scout Pro is an AI-powered streamlit application designed to help founders and startups identify potential venture capital firms and angel investors. By leveraging the Groq API and Llama 3.1, it generates curated lists of investors tailored to your specific industry, development stage, and geographic location.

Features
AI-Powered Search: Uses the Llama 3.1-8b-instant model for fast, context-aware results.

Customizable Filters:

Sector: Target specific niches (e.g., Generative AI, BioTech).

Geography: Filter by country or region.

Stage: Select from Pre-Seed to Private Equity.

Smart Configuration: Adjust the number of results to manage token usage.

Secure: API keys are handled via a password input field and are not stored permanently.

Responsive UI: Clean, wide-layout interface with custom styling.

Tech Stack
Python 3.8+

Streamlit: For the web interface.

Groq SDK: For ultra-low latency AI inference.
